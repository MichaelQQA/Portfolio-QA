**Tytuł**

Crash po dodaniu komentarza – komentarz nie zostaje dodany.

**Opis**

Po wypełnieniu pól oraz kliknięcia przycisku “Post Comment” komentarz nie zostaje dodany a strona się crashuje.

**Priorytet**

P1 - Krytyczny!

**Środowisko**

* Windows 11
* Chrome, Opera, FireFox, Microsoft Edge

**Kroki do Reprodukcji**

* Otwieramy stronę https://academybugs.com/
* Klikamy w find bugs
* Klikamy w produkt
* Wypełniamy pola Comment, Name, Email(jakąkolwiek wartością)
* Klikamy Post Comment

**Rzeczywisty Rezultat**

Po kliknięciu przycisku Post comment strona sie crashuje oraz komenatrz nie zostaje dodany.

**Oczekiwany Rezultat**

Komenatrz zostaje prawidłowo dodany.
